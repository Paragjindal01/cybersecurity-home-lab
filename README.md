# Cybersecurity Home Lab

This repository documents my cybersecurity home lab built on a Windows laptop.

## Laptop Specs
- 16GB RAM
- Intel i5
- 512GB SSD
- NVIDIA 1650
- 210GB free storage at setup

## Lab Stack
- Windows host
- WSL2 Ubuntu
- Git + GitHub
- VirtualBox
- Kali Linux VM
- Ubuntu Server VM
- Docker Desktop
- OWASP Juice Shop

## Goals
- Learn Linux
- Practice networking
- Learn Nmap and enumeration
- Build web security basics
- Document notes, scripts, and screenshots
